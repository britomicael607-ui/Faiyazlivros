# # Faiyazlivros 📚🖤

Catálogo supernatural de livros, mangás e HQs com estilo inspirado em Supernatural.

## 🎨 Características

- **Design Supernatural**: Tema escuro com paleta de cores vermelha, preta e azul anjo
- **Filtros Dinâmicos**: Filtre por categoria (Mangás, HQs, Livros) ou gênero (Terror, Ação, Romance)
- **Busca em Tempo Real**: Pesquise por título ou artefato
- **Interface Responsiva**: Grid adaptável para diferentes tamanhos de tela
- **Catálogo Personalizado**: Acervo de mangás com preços em Real (R$)

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Use os filtros para explorar o catálogo
3. Pesquise títulos específicos na barra de busca
4. Clique nos itens para ver detalhes (funcionalidade futura)
